About NodeGit

Asynchronous native Node bindings to the libgit2 project.

Version 0.1.0 | GitHub repository

History:

Originally started in 2011 as a project by TJ Fontaine, current Node lead, as an FFI-binding to libgit2. Tim Branyen then forked the project and migrated it to use native wrapper bindings, which would increase performance.

NodeGit was then led by Michael Robinson, who stabilized much of the core binding logic for the limited API exposed. Nick Kallen came on much later and automated a majority of the native code.

After benefiting from so many contributors, NodeGit is maturing and is slowly becoming a full suite of automated 1:1 bindings to libgit2.
